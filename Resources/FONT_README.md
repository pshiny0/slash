# Font Setup Required

The TANTangkiwood-Display font is already included in the Resources folder and configured in the project.

The font is referenced in:
- SplashView.swift (size 72)
- AuthView.swift (size 48) 
- HomeView.swift (size 32)
- AnalyticsView.swift (size 32)
- SharedView.swift (size 32)
- ProfileView.swift (size 32)

Font file: TANTangkiwood-Display.ttf
Font name used in code: "TANTangkiwood-Display"

If the font is not available, the app will fall back to the system font automatically.
